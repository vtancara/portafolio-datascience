Fact-Checker de Patrimonio Cultural Boliviano

Agentes Inteligentes con CrewAI

Laboratorio 4 — Módulo 10: NLP

Verónica Tancara

El problema

En redes sociales (especialmente TikTok), creadores de contenido atribuyen danzas bolivianas a otros países sin evidencia histórica.

Ejemplos frecuentes:

  • "La Morenada es peruana"
  • "La Diablada se originó en Puno"
  • "El Caporal es una danza peruana"
Pregunta: ¿Se puede automatizar la verificación de estos claims con agentes LLM?

La solución: sistema multi-agente

Agente 1: Investigador Cultural
Busca fuentes en internet (UNESCO, medios, académicos) sobre el origen de la danza
Agente 2: Verificador de Hechos
Contrasta fuentes, evalúa credibilidad, identifica consenso histórico
Agente 3: Redactor de Veredicto
Emite veredicto: VERDADERO / FALSO / PARCIALMENTE CIERTO

Configuración del LLM y herramientas

from crewai import Agent, Task, Crew, Process, LLM
from crewai.tools import BaseTool

# LLM: Google Gemini vía API
llm = LLM(
    model="gemini/gemini-2.5-flash",
    api_key=os.getenv("GOOGLE_API_KEY"),
)

# Herramienta de búsqueda web (Serper API)
class SerperSearchTool(BaseTool):
    name: str = "Buscar en internet"
    description: str = "Busca información en internet sobre un tema."

    def _run(self, query: str) -> str:
        response = requests.post(
            "https://google.serper.dev/search",
            json={"q": query, "gl": "bo", "hl": "es", "num": 10},
            headers={"X-API-KEY": os.getenv("SERPER_API_KEY")},
        )
        return "\n".join([r['title'] + ": " + r['snippet']
                          for r in response.json().get("organic", [])])

Definición de agentes

# Agente 1: busca fuentes reales en internet
investigador = Agent(
    role="Investigador de Patrimonio Cultural",
    goal="Buscar evidencia sobre el origen de la danza.",
    backstory="Especialista en folklore andino...",
    tools=[search_tool],  # Tiene acceso a búsqueda web

# Agente 2: contrasta y evalúa credibilidad
verificador = Agent(
    role="Verificador de Hechos Culturales",
    goal="Contrastar fuentes y emitir evaluación.",
    backstory="Fact-checker profesional...",
    llm=llm,  # Solo razonamiento, sin herramientas
)

# Agente 3: emite veredicto estructurado
redactor = Agent(
    role="Redactor de Veredicto",
    goal="Emitir veredicto: VERDADERO / FALSO.",
    backstory="Editor de agencia de fact-checking...",
    llm=llm,
)

Tareas y ejecución

# Cada claim pasa por los 3 agentes secuencialmente
def verificar_claim(claim: str):
    tarea_investigacion = Task(
        description=f'Busca en internet sobre: "{claim}"',
        agent=investigador,
    )
    tarea_verificacion = Task(
        description=f'Verifica: "{claim}" con las fuentes.',
        agent=verificador,
    )
    tarea_veredicto = Task(
        description=f'Emite veredicto para: "{claim}"',
        agent=redactor,
    )

    crew = Crew(
        agents=[investigador, verificador, redactor],
        tasks=[tarea_investigacion, tarea_verificacion, tarea_veredicto],
        process=Process.sequential,  # Uno tras otro
    )
    return crew.kickoff()  # Ejecuta la cadena completa

Claims verificados

El claim 4 (Pujllay) es un control — demuestra que el sistema no dice "FALSO" a todo.

Claim 1: La Morenada

FALSO

Evidencia clave:

  • UNESCO reconoce la Morenada en el Carnaval de Oruro (Bolivia)
  • Ley 135 de 2011 de Bolivia la declara Patrimonio Cultural e Inmaterial
  • BBC Mundo describe el origen como "disputado", no exclusivo de Perú

Fuentes: UNESCO, BBC Mundo, Wikipedia, UCSP

Claim 2: La Diablada

FALSO

Evidencia clave:

  • Wikipedia y múltiples fuentes sitúan el origen en Oruro, Bolivia
  • Historiador peruano reconoce que llegó a Puno en 1918 desde Bolivia
  • Infobae y La República la describen como de origen altiplánico compartido

Fuentes: Wikipedia, Infobae, La República, PuntoEdu PUCP

Claim 3: El Caporal

FALSO

Evidencia clave:

  • Creada por los hermanos Estrada Pacheco en La Paz, Bolivia, en 1969
  • La Cámara de Diputados de Bolivia celebra oficialmente su origen boliviano
  • Ninguna fuente confiable la atribuye como creación peruana

Fuentes: Wikipedia, Cámara de Diputados Bolivia, Sayacaporal.com

Claim 4: El Pujllay (control)

VERDADERO

Evidencia clave:

  • UNESCO inscribió "El Pujllay y el Ayarichi" como Patrimonio Inmaterial
  • Origen en la cultura Yampara de Chuquisaca, Bolivia
  • Todas las fuentes coinciden — no hay controversia

Fuentes: UNESCO, Wikipedia, Parlamento Andino

Reflexiones

Fortalezas de CrewAI: Orquestación transparente, integración con APIs, resultados estructurados y reproducibles.
Debilidades: Alto consumo de tokens, dependencia de la calidad del LLM, sin verificación de veracidad de fuentes.
Escenarios reales: Defensa del patrimonio cultural, monitoreo de desinformación en redes, educación sobre folklore boliviano.

Conclusiones

  • Un sistema multi-agente puede verificar claims culturales automáticamente con evidencia real
  • La calidad del LLM es determinante — modelos pequeños fallan en la interpretación
  • La Morenada, Diablada y Caporal son danzas bolivianas con evidencia verificable (UNESCO, leyes, académicos)
  • Estas herramientas amplifican la capacidad humana de verificación, no la reemplazan
